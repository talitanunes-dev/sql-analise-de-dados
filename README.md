# Consultas SQL para Análise de Dados

Repositório com consultas SQL simples, usadas para treinar lógica e análise de dados.

## O que tem aqui:
- SELECT básico  
- Filtros (WHERE)  
- Agrupamento (GROUP BY)  
- Agregações (SUM, COUNT)  
- Ordenação  
- JOIN entre tabelas  

Essas queries simulam cenários reais de análise, como faturamento, contagem de vendas e relação entre clientes e compras.
